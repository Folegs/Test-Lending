Our Requirement

Front end - HTML5 and CSS3 are main technology of the UI code (CSS3 can replace JS in many cases).

Resolutions ready - link - need to include all desktops, including large screens and resolutions.

Pixel-Perfect levelof UI - must be accurate on pixel level to PSD design.

Code - classes, IDis and scripts names - regured to be named properly - for future editing.

Cannot be any copyright mark, link, name, signature, email, images in any of the code files, files names, project images and any other project asset.

Sliders with layers & float text required to fit for any responsive layout.

The developer cannot use any modules and plugins which are not free to use & with free license of use.

1 element for all layouts - Header, Footer, Body, Menu, fields & forms - cannot have duplicate elements for desktop and mobile (you required to do 1 elements which will fit to all layouts properly).
